## Результаты стабилизации

### SA-01.2 — MCP

- `/mcp` перенаправляется относительным `307` на `/mcp/`;
- redirect не зависит от внутренней HTTP-схемы reverse proxy;
- DNS-rebinding protection остаётся включённой;
- произвольные Host и Origin отклоняются.

### SA-01.3 — единичный внешний поиск

- `/api/company-intelligence` собирает внешний пакет источников один раз;
- глубокий анализ официального сайта повторно использует уже собранный пакет;
- regression-тест фиксирует число обращений к поисковому контуру.

### SA-01.4 — независимая классификация

- `query_kind` хранит намерение запроса;
- `result_kind` описывает найденный документ;
- `source_class` определяется по фактическому домену;
- конфликты маркируются `ambiguous`, отсутствие данных — `unknown`.

### SA-01.5 — жизненный цикл evidence

- поисковая выдача создаёт только `discovery_hint`;
- выбранный URL может стать `source_candidate`;
- `evidence` создаётся только после загрузки первичного документа;
- evidence содержит URL документа, дату доступа и цитату;
- ошибка загрузки не повышает состояние и evidence strength;
- одни поисковые сниппеты не дают статус `complete` и не повышают fallback score.
