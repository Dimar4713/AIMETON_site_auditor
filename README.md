# AIMETON Site Auditor

Экспериментальный модуль экономической разведки AIMETON. Исследует цифровое присутствие компании, выявляет экономические сигналы, квалифицирует коммерческую возможность, подбирает подходящий AI-инструмент и формирует пакет действия для корректного персонального контакта.

## Принцип РПТК7 «Эссетер»

Публичная техническая формулировка контура:

1. обнаружение экономических сигналов;
2. квалификация и формирование коммерческого контура;
3. подготовка и передача коммерческой возможности.

Исходная функциональная метафора «нюхач → загон → поднос» используется только как пояснение происхождения принципа и не является названием программных модулей.

## Что возвращает анализ

- профиль компании и наблюдаемые доказательства;
- экономические сигналы с уровнем уверенности;
- гипотезу возможной потери спроса, времени или качества;
- оценку коммерческой возможности от 0 до 100;
- рекомендуемый AI-инструмент;
- дополнительные варианты автоматизации;
- гипотезу о роли лица, принимающего решение;
- демонстрационный сценарий;
- проект первого персонального сообщения;
- следующий проверяемый шаг;
- ограничения и неподтверждённые предположения.

Аудитор не утверждает наличие реальных потерь без данных компании и не предназначен для массового спама. Его задача — подготовить доказательную коммерческую возможность для ручной проверки и осмысленного делового контакта.

## Запуск в Replit

1. Импортируйте репозиторий в Replit.
2. Добавьте Secret `ROUTERAI_API_KEY`.
3. При необходимости задайте `ROUTERAI_MODEL` (по умолчанию `openai/gpt-4o-mini`).
4. Выполните `pip install -r requirements.txt`.
5. Запустите `uvicorn app.main:app --host 0.0.0.0 --port 5000`.

RouterAI используется через OpenAI-совместимый endpoint `https://routerai.ru/api/v1`.

## Безопасность

- ключ хранится только в переменных окружения;
- SSRF-защита блокирует локальные и служебные IP;
- ограничены тип, размер и время загрузки страницы;
- LLM не получает право самостоятельно выполнять внешние действия;
- выводы разделяются на доказательства и гипотезы;
- автоматическая массовая рассылка не входит в разрешённый контур.

## Тесты

```bash
pytest -q
```

Включены воспроизводимые сценарии анализа цифрового ритейла, облачной инфраструктуры и логистики. Экспериментальная ветка также содержит контур динамического рендеринга страниц.
