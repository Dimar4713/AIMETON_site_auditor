# AIMETON Site Auditor

MVP AI-аудитора сайта: загружает главную страницу компании, определяет профиль бизнеса, предлагает 5–10 AI-агентов и продолжает консультацию в чате через RouterAI.

## Запуск в Replit

1. Импортируйте репозиторий в Replit.
2. Добавьте Secret `ROUTERAI_API_KEY`.
3. При необходимости задайте `ROUTERAI_MODEL` (по умолчанию `openai/gpt-4o-mini`).
4. Выполните `pip install -r requirements.txt`.
5. Запустите `uvicorn app.main:app --host 0.0.0.0 --port 5000`.

RouterAI используется через OpenAI-совместимый endpoint `https://routerai.ru/api/v1`.

## Безопасность

- ключ хранится только в переменных окружения;
- SSRF-защита блокирует локальные и служебные IP;
- ограничены тип, размер и время загрузки страницы;
- LLM не получает право самостоятельно выполнять внешние действия.
